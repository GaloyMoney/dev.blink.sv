Lookup invoice and verify if it was paid

query TransactionsByPaymentHash($paymentHash: PaymentHash!, $walletId: WalletId!) {
  me {
    defaultAccount {
      walletById(walletId: $walletId) {
        transactionsByPaymentHash(paymentHash: $paymentHash) {
          status
          direction
        }
      }
    }
  }
}
{
  "paymentHash": null,
  "walletId": null
}

if you have the invoice you can get the payment hash on: https://dev.blink.sv/decode

Can also do it more directly with these operations
